# 📦 Map Analyzer - ビルド不要!インストーラー使用ガイド

## 🎯 このガイドの対象者

- **Flutter SDKを持っていない方**

## 🚀 クイックスタート (3ステップ)

### ステップ1: 自分のOSを確認

あなたのパソコンのOSを確認してください:
- 🪟 **Windows** (Windows 10/11)
- 🍎 **Mac** (macOS)
- 🐧 **Linux** (Ubuntu, Debian, Fedora等)

### ステップ2: インストーラーをダウンロード

**GitHub Releases**から自分のOSに合ったインストーラーをダウンロード:

👉 **https://github.com/geomatsuyama/Maptag/releases**

**ダウンロードするファイル:**
- 🪟 Windows: `MapAnalyzer_Setup_v1.0.0_x64.exe` (25-30MB)
- 🍎 macOS: `MapAnalyzer_v1.0.0_macOS.dmg` (20-25MB)
- 🐧 Linux: `MapAnalyzer-1.0.0-x86_64.AppImage` (23-28MB) ← 推奨

### ステップ3: インストール

#### 🪟 Windows
1. ダウンロードした `.exe` ファイルをダブルクリック
2. セットアップウィザードに従う (2-3分)
3. 完了後、スタートメニューから「Map Analyzer」を起動

#### 🍎 macOS
1. ダウンロードした `.dmg` ファイルをダブルクリック
2. 開いたウィンドウで `Map Analyzer.app` を `Applications` フォルダにドラッグ
3. Launchpadから「Map Analyzer」を起動

#### 🐧 Linux
```bash
# ダウンロードしたファイルに実行権限を付与
chmod +x MapAnalyzer-1.0.0-x86_64.AppImage

# 実行
./MapAnalyzer-1.0.0-x86_64.AppImage
```

**たったこれだけ!Flutter SDKは一切不要です!**

## 📋 初回起動時の設定 (3分)

インストール後、初回起動時に以下の設定が必要です:

### 1. Mapillary APIキーを取得

**取得先**: https://www.mapillary.com/developer

**手順:**
1. Mapillary公式サイトでアカウント登録 (無料)
2. Developer Portalで新しいアプリケーションを作成
3. Client IDをコピー
4. Map Analyzerの設定画面に貼り付け

### 2. Gemini APIキーを取得

**取得先**: https://makersuite.google.com/app/apikey

**手順:**
1. Google AI Studioにアクセス
2. 「Create API Key」をクリック
3. APIキーをコピー
4. Map Analyzerの設定画面に貼り付け

### 3. 無料/有料モードを選択

**無料モード** (推奨・初心者向け):
- 1回の検索で最大100枚の画像メタデータ取得
- APIレート制限あり

**有料モード** (上級者向け):
- 1回の検索で最大200,000枚の画像メタデータ取得
- 大規模GIS分析用

## 💡 使い方 (基本編)

### 1. Mapillary画像を検索

**3つの検索方法:**
1. **地点+半径**: 「東京タワー周辺1km」のように検索
2. **矩形範囲**: 地図上で四角形を描いて範囲指定
3. **任意ポリゴン**: 複雑な形のエリアを指定

### 2. Gemini AIで画像分析 (オプション)

- 任意のプロンプトで画像を分析
- 例: 「この画像に写っている交通標識を説明して」
- バッチ処理で複数画像を一度に分析

### 3. データをエクスポート

**3つの形式:**
- **JSON**: GIS解析用 (QGIS、ArcGIS等で使用)
- **Excel**: スプレッドシート分析用
- **CSV**: 汎用データ形式

**エクスポート内容:**
- 位置情報 (緯度・経度・高度)
- 撮影情報 (日時・方位角)
- カメラ情報
- Mapillary情報
- AI分析結果 (オプション)

## 🔧 トラブルシューティング

### Windows: "Visual C++ Redistributableが必要"エラー

**解決策:**
1. 以下をダウンロード: https://aka.ms/vs/17/release/vc_redist.x64.exe
2. インストール
3. Map Analyzerを再起動

### macOS: "開発元が未確認のため開けません"エラー

**解決策:**
1. 「システム環境設定」→「セキュリティとプライバシー」を開く
2. 「このまま開く」をクリック

または、ターミナルで:
```bash
xattr -cr /Applications/Map\ Analyzer.app
```

### Linux: "実行できません"エラー

**解決策:**
```bash
chmod +x MapAnalyzer-1.0.0-x86_64.AppImage
```

### すべてのOS: APIキーエラー

**解決策:**
- Mapillary APIキーが正しいか確認
- Gemini APIキーが正しいか確認
- インターネット接続を確認

## 🎯 よくある質問 (FAQ)

### Q: Flutter SDKは必要ですか?
**A**: いいえ、不要です!ビルド済みインストーラーをダウンロードするだけです。

### Q: 無料で使えますか?
**A**: はい!アプリ自体は無料です。ただし、Mapillary APIとGemini APIは各サービスの利用規約に従います。

### Q: APIキーの取得は有料ですか?
**A**: 
- **Mapillary API**: 無料プランあり (制限あり)
- **Gemini API**: 無料プランあり (月間制限あり)

### Q: オフラインでも使えますか?
**A**: 一部機能のみ。Mapillary検索とGemini AI分析にはインターネットが必要です。

### Q: アンインストール方法は?
**A**:
- **Windows**: コントロールパネル → プログラムと機能
- **macOS**: Applications フォルダから削除
- **Linux**: ファイルを削除するだけ (AppImageの場合)

### Q: データはどこに保存されますか?
**A**: すべてローカルに保存されます。クラウド同期はありません。

### Q: 商用利用できますか?
**A**: はい。MITライセンスなので、商用利用可能です。
